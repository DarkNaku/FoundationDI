# status 명령어 - 진행 상황 확인

plan.md의 현재 진행 상황과 테스트 상태를 표시합니다.

## 표시 내용

1. **Git 상태**: 현재 브랜치, 미커밋된 변경사항
2. **진행률**: plan.md 기반 완료/미완료 테스트 수
3. **테스트 상태**: 현재 테스트 실행 결과
4. **최근 커밋**: 최근 커밋 정보
5. **다음 작업**: 다음 예정 테스트

## 실행 결과 예

```
📊 PoolService 개선 진행 상황

┌─────────────────────────────────────────┐
│ P0 (Critical):    1/2 ██░ 50%           │
│ P1 (Important):   0/3 ░░░ 0%            │
│ P2 (Enhancement): 0/5 ░░░ 0%            │
│ ─────────────────────────────────────── │
│ 전체:              1/10 ██░░░ 10%      │
└─────────────────────────────────────────┘

✅ 완료된 테스트:
└─ P0-1: shouldReturnNullWhenGameObjectDestroyed

⏳ 다음 테스트:
└─ P0-1: shouldHandleNullGameObjectInGet

📝 최근 커밋:
└─ [BEHAVIORAL] GameObject null-safe 처리
```

## 사용 예

```
/status
```

다음 단계: `/go` (다음 테스트 시작)
