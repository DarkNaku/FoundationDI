# red 명령어 - RED 단계 상세 실행

현재 작업 중인 테스트의 RED 단계를 상세하게 진행합니다.

## 실행 내용

1. **테스트 파일 확인**: 현재 테스트 파일 위치 확인
2. **테스트 코드 작성**: 실패하는 테스트 작성
3. **테스트 실행**: NUnit/Unity Test Framework로 실행
4. **실패 메시지 표시**: 왜 실패하는지 명확하게 표시

## RED 단계 원칙

✅ **반드시 지켜야 할 규칙**:
- 테스트는 실패해야 함 (❌)
- 프로덕션 코드를 수정하면 안 됨
- 테스트 코드만 작성
- 한 번에 하나의 행동만 검증

✅ **좋은 테스트 특징**:
- Arrange-Act-Assert 패턴 사용
- 테스트 이름이 테스트 목적을 명확히 함
- 하나의 개념만 테스트
- 테스트가 실패하는 이유가 명확함

## 사용 예

```
/red
```

다음 단계: `/green` (최소 코드로 통과)
